## Giới thiệu

Ứng dụng này cho phép người dùng xem danh sách các bài viết từ API [jsonplaceholder](https://jsonplaceholder.typicode.com/posts) và thêm bài viết mới thông qua component `CreatePost`. Ứng dụng sử dụng Redux để quản lý trạng thái, giúp thao tác dữ liệu một cách hiệu quả.

## Hướng dẫn Cài Đặt và Chạy Ứng Dụng

1. **Clone Repository**: Clone repository về máy:

   ```bash
   git clone https://github.com/minhtu123ab/Test-BlueOc.git
   ```

2. **Chuyển vào Thư Mục Dự Án**: Chuyển vào thư mục dự án:

   ```bash
   cd task3
   ```

3. **Cài Đặt Các Gói Cần Thiết**: Chạy lệnh sau để cài đặt tất cả các gói cần thiết:

   ```bash
   npm install
   ```

4. **Chạy Ứng Dụng**: Để chạy ứng dụng, sử dụng lệnh:
   ```bash
   npm run dev
   ```
   Mở trình duyệt và truy cập `http://localhost:5173` để xem ứng dụng.

## Các Tính Năng

- **Hiển Thị Bài Viết**: Danh sách các bài viết được tải từ API và hiển thị trong 1 bảng của ứng dụng.
- **Thêm Bài Viết**: Người dùng có thể thêm bài viết mới thông qua một form.
- **Quản Lý Trạng Thái**: Ứng dụng sử dụng Redux để quản lý trạng thái bài viết và các thao tác liên quan.
- **Thông Báo Trạng Thái**: Hiển thị thông báo cho người dùng khi thực hiện các thao tác như tải dữ liệu hay thêm bài viết.
